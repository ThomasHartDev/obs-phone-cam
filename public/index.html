    Set the Browser Source Width and Height to match your canvas: 1080×1920 for vertical (TikTok/IG), 1920×1080 for landscape (YouTube). Then right-click the source → Transform → Fit to screen. A wide box with black bars means the source size doesn’t match the canvas. The URL is plain http (OBS can’t open the https one).

    3. Use it as a camera device

    In OBS, right side under Controls, click Start Virtual Camera.

    Your phone now shows up as the camera named “OBS Virtual Camera” in Zoom, Teams, Discord, Chrome, or any app’s camera list. (OBS needs to stay open while you use it.)
